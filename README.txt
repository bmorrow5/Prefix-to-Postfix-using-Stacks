This program is designed to convert a prefix to postfix expresssion using a stack data structure.
It can be run from the command line


1) To run the program enter the folder location: Prefix to Postfix using Stacks


2) After you input the folder location enter the following with input and output file locations:

java -jar Project01.jar [inputFile] [outputfile]

EXAMPLE: 
C:\Users\brand\Documents\BMorrow>java -jar Project01.jar
